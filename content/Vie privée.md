Ce jardin numérique est propulés par Quartz. 

Or…

Quartz **n’utilise pas de cookies**. 

Cependant, il utilise **`localStorage`** et **`sessionStorage`** dans certains scripts JavaScript pour stocker des préférences localement dans le navigateur, notamment dans :

- `checkbox.inline.ts` : pour mémoriser l'état de cases à cocher.
    
- `darkmode.inline.ts` : pour mémoriser la préférence de thème (mode clair/sombre).
    
- `explorer.inline.ts` : pour l'état du panneau de navigation.
    
- `graph.inline.ts` : pour des interactions graphiques.
    

Cela signifie que Quartz **n'installe pas de traceurs tiers ni de cookies traditionnels** (soumis au RGPD), mais utilise le stockage local pour améliorer l'expérience utilisateur.
